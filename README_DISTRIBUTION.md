# CoreCoin (CORE) - Official Wallet v1.0

Welcome to the CoreCoin Network. This is the official Windows distribution for the CoreCoin Core wallet.

## Quick Start Instructions

1. **Launch the Wallet:**
   Double-click `CoreCoin-qt.exe`.

2. **Network Connection:**
   The wallet is pre-configured to automatically connect to our global seed nodes. If you see "Connecting to peers..." for more than 2 minutes, follow these steps:
   - Go to **Help** -> **Debug Window** -> **Console**.
   - Type: `addnode "136.116.228.213:9777" onetry`
   - Connectivity bars should appear in the bottom right corner.

3. **Mining CORE:**
   CoreCoin uses a Proof-of-Work consensus. To support the network and earn CORE:
   - Open the Console (Help -> Debug Window -> Console).
   - Type: `generatetoaddress 1 <your_address>`

## Units
- 1 CORE = 100,000,000 sat
- milliCORE (mCORE) = 0.001 CORE
- microCORE (uCORE) = 0.000001 CORE
